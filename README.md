# 🏹 OldDragon -- RPG

![RPG Banner](https://img.shields.io/badge/RPG-Python-blue?style=for-the-badge)

Bem-vindo ao **OldDragon -- RPG**, um jogo de RPG totalmente feito em Python que conta com rolagem de dados aleatória e criação dos atributos dos persoangens!  
Divirta-se criando seu personagem no terminal.

---

## 🎮 Sobre o Jogo

No **OldDragon -- RPG**, você poderá:

- Criar seu personagem e atribuir seu estilo e atributos.  

O jogo foi desenvolvido para **diversão e aprendizado de Python**, utilizando programação orientada a objetos, listas e dicionários.
